# Endpoint and Workstation Policy

| Field | Value |
|---|---|
| Policy ID | EWP-001 |
| Version | 1.0 |
| Effective Date | March 9, 2026 |
| Owner | Security Officer |

## Approved Device Posture

- Company-managed devices are preferred
- BYOD allowed only with security controls and management approval

## Minimum Security Requirements

- Full disk encryption enabled
- Auto-patching enabled for OS and critical apps
- Screen lock timeout ≤ 5 minutes
- EDR/anti-malware active and up-to-date
- Admin rights minimized; no routine local admin use

## Prohibited Practices

- Storing PHI locally outside approved workflows
- Disabling endpoint protections
- Installing unapproved remote control software

## Remote Access

- Use approved VPN/ZTNA paths only
- MFA required for all privileged/administrative access

## Lost/Stolen Device

- Report within 1 hour
- Trigger session revocation, credential reset, and incident triage

## Home Office Controls

- Private workspace for sensitive activity
- Lock screen when unattended
- Prevent shoulder surfing/unauthorized viewing

## Evidence

- Endpoint compliance reports
- Patch posture reports
- Lost-device incident records
