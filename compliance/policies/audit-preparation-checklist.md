# Audit Preparation Checklist

## Readiness Gates

- [ ] Policy library complete and approved
- [ ] Control ownership/evidence matrix current
- [ ] Risk assessment completed and current quarter reviewed
- [ ] Risk register updated with treatment status
- [ ] Vendor/BAA registers up to date

## Technical Evidence

- [ ] CloudTrail trail active and immutable logging configured
- [ ] GuardDuty detector enabled and triage workflow evidenced
- [ ] AWS Config recorder/rules enabled with compliance exports
- [ ] Security Hub standards enabled and findings triaged
- [ ] Backup plan evidence and restore test evidence available
- [ ] Vulnerability scans and remediation SLA evidence available
- [ ] Access review records (quarterly) complete
- [ ] Security training completion evidence complete

## SDLC / Change Evidence

- [ ] CI gate outputs retained for sample period
- [ ] PR approvals and CODEOWNERS evidence available
- [ ] Migration safety and rollback evidence captured

## SOC 2 Type I / II

- [ ] Type I design walkthrough package complete
- [ ] Type II observation window evidence organized by control
- [ ] Auditor PBC request tracker prepared

## HIPAA Readiness

- [ ] Security Rule safeguards mapped and evidenced
- [ ] Breach procedure and incident records complete
- [ ] BAA obligations demonstrated
